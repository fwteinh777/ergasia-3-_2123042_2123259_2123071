Ο "server" δημιουργεί έναν socket listener στη θύρα 8080.
Για κάθε πελάτη (customer), το πρόγραμμα κάνει fork(). 
Η γονική διεργασία (server) κάνει accept() και εξυπηρετεί τη σύνδεση του πελάτη.
Η θυγατρική διεργασία λειτουργεί ως client, συνδέεται στον server και στέλνει παραγγελίες (orders).
Ο server διαθέτει έναν "κατάλογο" (catalog) προϊόντων με συγκεκριμένη ποσότητα 
και τιμή. Όταν εξαντλείται το απόθεμα ενός προϊόντος, οι επόμενες παραγγελίες αποτυγχάνουν.
Μετά την ολοκλήρωση όλων των παραγγελιών, ο server εμφανίζει στατιστικά στοιχεία (πλήθος επιτυχημένων/αποτυχημένων παραγγελιών, κέρδος, κλπ.).


Δομή Αρχείων
1) catalog.h      - Αρχείο κεφαλίδας που περιέχει δηλώσεις & ορισμούς (π.χ. struct catalog, #define).
2) server.c       - Το κύριο αρχείο κώδικα που περιλαμβάνει:
Δημιουργία καταλόγου (createCatalog)
Διαχείριση πελατών (handleCustomer)
Συνάρτηση main με τον server και τους forked clients.
3) README.txt     - Αυτό το αρχείο, συνοπτικές οδηγίες.
4) Makefile       - Κανόνες για τη μεταγλώττιση και τον καθαρισμό των παραγόμενων αρχείων.


Μεταγλώττιση & Εκτέλεση
Απαιτείται gcc (σε Linux/WSL/MacOS).
Από γραμμή εντολών (terminal), πληκτρολογήστε:
    make        (για μεταγλώττιση)
    ./server    (για εκτέλεση)
Το πρόγραμμα ανοίγει θύρα 8080, δημιουργεί forked πελάτες και κάνει accept για κάθε έναν. 
Στο τέλος εκτυπώνει στατιστικά.
